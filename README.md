Main analysis featured on manuscript are located in `analysis`

The CCMs were done using the following

- lat: 41
- lon: 51
Total Runs: 41\*51\*3 = 6273 


SLP runs were conducted and not shown

FOR SLP:
- lat: 33
- lon: 41
Total Runs: 33\*41\*3 = 4059
